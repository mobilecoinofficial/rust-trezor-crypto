Module trezor_crypto_lib::ed25519

source · [−]
Expand description

Curve25519 operations over sha512 (default), keccak, and sha3.

These functions are ABI compatible with those provided by donna, with the addition of a dalek_ prefix to allow linking both implementations for testing.

keccak

ed25519 API using keccak512 signatures, equivalent to ed25519-donna APIs generated with a custom keccak512 hasher (see tests/ed25519-keccak.c)

sha3

ed25519 API using sha3 signatures, equivalent to ed25519-donna APIs generated with a custom sha3 hasher (see tests/ed25519-sha3.c)
